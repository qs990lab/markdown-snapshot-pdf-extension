# 上付き・下付き文字テスト

## 化学式
- 水: H~2~O
- 二酸化炭素: CO~2~
- 硫酸: H~2~SO~4~

## 数学式
- アインシュタインの質量エネルギー等価性: E = mc^2^
- 指数: x^2^ + y^2^ = z^2^
- 複雑な指数: 2^n+1^

## 組み合わせ
- 複雑な化学式: Ca(OH)~2~ + H~2~SO~4~ → CaSO~4~ + 2H~2~O
- 数学と化学の組み合わせ: x^2^ + H~2~O = solution

## 通常のテキスト
これは通常のテキストです。~これは下付きではありません~ ^これは上付きではありません^

## 正しい記法
- 下付き文字: H~2~O（水）
- 上付き文字: E = mc^2^（相対性理論）
